指定站点名、作者以及站点描述

编辑根目录下_config.yml文件

其中title表示站点标题，
description表示站点描述，
author表示站点主人，
language表示站点语言，这里通过themes/landscape/languages目录下的文件名来确认，这里使用中文即由zh-CN表示，landscape表示默认主题
timezone表示站点时区，默认为当前电脑所在时区，可参考时区列表自行设定

安装Next主题

站点根目录下执行命令git clone https://github.com/theme-next/hexo-theme-next themes/next进行安装

配置Next主题

编辑themes/next/目录下_config.yml文件，查找scheme,会发现有四种不同的

这里注释掉Muse,启用Gemini,即博客现在使用的主题。

首页、分类、归档、标签设置

1.编辑themes/next/目录下_config.yml文件，查找menu,

home表示首页，tags表示标签，categories表示分类，archives表示归档
去掉前面的#即使用对应的功能，其中||之前的值是目标链接，之后的是分类页面的图标，图标名称来自于FontAwesome icon。若没有配置图标，默认会使用问号图标。

2.hexo根目录下执行命令hexo new page "categories",之后会在source文件夹中生成一个categories文件夹，里面有一个index.md文件，编辑此文件：

设置type为"categories",tags,archives配置类似。

设置站点创建时间

打开next主题目录下_config.yml,查找since

去掉since前的注释，如果不设置则显示当前年份。

设置头像

打开next主题目录下_config.yml,查找avatar

url表示图片位置，
若将图片放置themes/next/source/images/下，则url为/images/avater.png
若将图片放置hexo跟目录下的source/uploads目录下则url为/uploads/avater.png
rounded为true表示头像显示为圆形
rotated为true表示鼠标移动到头像时否可旋转

网站图标

1.制作网站图标，参考图标素材网站：iconfont,easyicon
2.打开next主题目录下_config.yml,查找favicon

修改small和medium的路径为下载的图标路径

文章添加评论

1.注册Leancloud
2.注册完后添加应用app
3.进入应用app，点击设置在基本信息中你会发现应用Keys,点进去拿到AppID和AppKey
4.编辑themes/next/_config.yml文件，找到valine,将enable置为true，将保存的AppID和AppKey分别赋值给appid和appkey
5.进入Leancloud app应用的安全中心，将博客域名填入Web 安全域名，完工
