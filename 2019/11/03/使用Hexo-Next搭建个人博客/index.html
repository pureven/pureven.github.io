创建GitHub Pages

使用Jekyll创建GitHub Pages网站
您可以使用Jekyll在新的或现有存储库中创建GitHub Pages站点。官网给出的方法是使用Jekyll创建GitHub Pages网站

或根据如下流程创建：
1.点击New respository

Repository name 必须是 you_name.github.io

点击 “Create respository”,完成创建。 然后点击Settings，向下拉到最后有个GitHub Pages，Custom domain填写已注册的域名，会变成下面这样：

安装Hexo

下载并安装Node.js 官网下载地址

安装Hexo可参考官网
或根据如下步骤：

1.新建用于Hexo项目的目录，比如pureven，然后执行命令npm i hexo-cli进行安装

2.安装完成后，先通过hexo init初始化hexo,然后通过hexo g生成静态网页，通过hexo s --debug开启服务，
然后浏览器输入localhost:4000即可查看默认样式的博客啦。

其他命令可参加官网命令说明
