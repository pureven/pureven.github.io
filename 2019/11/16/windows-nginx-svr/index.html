将NGINX注册为Windows系统服务

发表于 2019-11-16 更新于 2020-04-13 分类于 开发者手册 Valine：
本文字数： 1.4k 阅读时长 ≈ 1 分钟

NGINX安装

  • 首先在nginx官网下载到的压缩包，目前稳定版本为nginx-1.16.1
  • 解压后放入D盘
  • 运行NGINX，直接双击nginx.exe即可

问题

通过双击运行nginx后，有几点不便：

  1. 启停不方便，修改配置后启停nginx需要cmd操作
  2. 无法开机自启动

解决方案

将nginx加入到windows服务
下面介绍使用winsw将nginx加入到windows服务的操作流程：

  1. 下载winsw,目前稳定版为winsw-v2.2.0，本例下载的为WinSW.NET2.exe
  2. 将WinSW.NET2.exe重命名为nginxsvr.exe并放入与nginx.exe相同目录下
  3. 新建配置文件nginxsvr.xml，参考这里进行配置，本例配置信息为：
  4. cmd终端运行nginxsvr.exe install
